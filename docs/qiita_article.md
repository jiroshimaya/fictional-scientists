# 架空の科学者を１万人生成

## はじめに

音楽カバー動画のいわゆる「〇〇で歌ってみたシリーズ」を作るとき、実在の人物の声やビジュアルを使うと著作権・肖像権が問題になります。

そこで、**完全に架空の科学者を大量に生成**することにしました。名前・研究分野・業績・肖像画をすべて AI で作るので、著作権も肖像権も気にする必要がありません。

生成するデータは以下の４要素です。

- 名前（姓・名・読み仮名）
- 主な分野・業績・研究内容
- 肖像画生成プロンプト
- 肖像画（PNG）

## 方針

### 1. ChatGPT と相談してクォータ（種）を設計する

いきなり１万人分をランダムに生成すると、現代の欧米男性ばかりになりがちです。そこで先に「時代 × 国籍 × 分野 × 性別」の組み合わせとそれぞれの人数（クォータ）を設計しました。

ChatGPT と対話しながら以下を決定しました。

- **時代区分**：古代前期（紀元前400〜紀元前1）、古代後期、中世、近世、近代、現代 など
- **国籍・地域**：古代ギリシア、戦国・秦漢中国圏、古代インド圏、近代日本、現代アフリカ など多数
- **研究分野**：光学、力学、天文学、化学、生物学、数学 など
- **性別**：男性・女性・ノンバイナリー／その他

これを CSV（クォータマスター）として保存し、各行を `count` の数だけ展開することで ID 付きの一覧（約１万行）を生成しました。

```
era_name,field,nationality,count
近代,量子力学,日本,10
近代,量子力学,ドイツ,15
…
```

展開後の ID 形式：`近代__物理学__日本__0001`

### 2. 名前・プロフィールを並列生成する

クォータ展開済み CSV を入力として、名前とプロフィールを OpenAI API で生成します。

**名前生成**（`gpt-4.1-mini`）

各行の時代・国籍・性別情報をプロンプトに渡し、姓・名・読み仮名を JSON Schema で構造化出力します。重複チェック（文字列類似度 ≥ 90%）も実施しています。

```
出力フィールド: 名前, 姓, 名, ナマエ, セイ, メイ
```

**プロフィール生成**（`gpt-4.1-mini`）

名前と属性情報をもとに以下を生成します。

```
出力フィールド: 生年, 没年, 主な分野, 業績・受賞歴,
               研究内容（要約）, 研究内容（詳細）
```

### 3. 時代に合った肖像画プロンプトを生成する

プロフィール JSONL を入力として、`gpt-4.1-mini` で画像生成用プロンプトを作成します。

ポイントは**時代考証**です。古代の科学者には油彩画や彫刻、近代の科学者にはモノクロ写真風など、時代に合ったスタイルをプロンプトに反映させます。

```json
{
  "portrait_prompt": "Black and white photograph...",
  "negative_prompt": "color photo, modern...",
  "style_note": "Early 20th century monochrome portrait"
}
```

### 4. 肖像画を生成する

肖像画プロンプト JSONL を入力として `gpt-image-1` で 1024×1024 の PNG を生成し、`data/portraits/` に保存します。

### 5. 全体を１ファイルに纏める

各フェーズの出力（名前 CSV・プロフィール JSONL・肖像画 JSONL）をマージして、最終的な `fictional_scientists.jsonl` を生成します。

## パイプライン全体像

```
クォータマスター CSV
    ↓ expand_quota.py      — 属性の組み合わせを展開 → ID付きCSV
    ↓ generate_name.py     — 名前・読み仮名を生成
    ↓ generate_profile.py  — プロフィールを生成
    ↓ generate_portrait_prompt.py  — 肖像画プロンプトを生成
    ↓ generate_images.py   — 肖像画 PNG を生成
最終 JSONL（fictional_scientists.jsonl）
```

taskipy でワンコマンド実行できます。

```bash
DIR=data/sample/two uv run task generate
```

## まとめ

- ChatGPT と対話してクォータを設計することで、時代・地域・性別のバランスが取れた１万人分のデータが得られた
- 名前・プロフィール・肖像画プロンプト・画像をすべて OpenAI API で生成するパイプラインを構築
- 時代考証をプロンプトに組み込むことで、それぞれの時代らしい肖像画スタイルが実現できた
- 著作権・肖像権フリーな架空人物データセットとして、様々なコンテンツ制作に活用できる
